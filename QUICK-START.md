# UXNitro Quick Start Guide

Welcome to UXNitro! This guide will help you get your fast, modern website up and running in minutes.

---

## 📦 Installation (2 minutes)

### Method 1: Upload Through WordPress
1. Log into your WordPress admin
2. Go to **Appearance → Themes**
3. Click **Add New**
4. Click **Upload Theme**
5. Choose the UXNitro zip file
6. Click **Install Now**
7. Click **Activate**

### Method 2: Manual Upload
1. Upload `UXNitro-master` folder to: `/wp-content/themes/`
2. Go to WordPress admin → **Appearance → Themes**
3. Find **UXNitro** and click **Activate**

After activation, you'll see the UXNitro welcome page with:

## ⚡ Optimize Performance (5 minutes)

Go to **Customize → UXNitro Performance**

### Quick Optimization (Safe for 95% of sites)

Enable these settings:

**WordPress Cleanup:**
- ✅ Disable Emojis
- ✅ Disable Dashicons for Guests
- ✅ Disable XML-RPC
- ✅ Disable Pingbacks
- ✅ Disable Shortlink
- ✅ Disable Generator Meta Tag
- ✅ Disable Query Strings from Static Files
- ✅ Disable Comment Reply Script
- ✅ Disable jQuery Migrate

**Font Optimization:**
- ✅ Font Display Swap

**Media Optimization:**
- ✅ Enable Lazy Load for Images
- ✅ Enable Lazy Load for Iframes

**Database & Background Tasks:**
- Set Heartbeat Frequency to **60 seconds**

**Admin Optimization:**
- ✅ Disable Welcome Panel
- ✅ Disable Update Nags for Non-Admin Users

Click **Publish** to save!

## 🔧 Common Tasks

### Create a Homepage
1. Create a new page (Pages → Add New)
2. Design with Elementor or Gutenberg
3. Go to **Settings → Reading**
4. Set as static homepage

### Install Elementor
1. Plugins → Add New
2. Search "Elementor"
3. Install and Activate
4. Start building pages!

### Set Up Menu
1. Appearance → Menus
2. Create new menu
3. Add pages/links
4. Assign to "Primary" location

### Add Widgets
1. Appearance → Widgets
2. Add to sidebars or footer areas
3. Configure each widget

## 🚀 Speed Tips

### Essential Plugins
Install these for maximum performance:
- **LiteSpeed Cache** (free) - Best for LiteSpeed servers
- **WP Rocket** (paid) - Best overall caching
- **Elementor** (free) - Page builder

### Image Optimization
- Compress images before uploading
- Use WebP format when possible
- Enable lazy loading (already done in Performance panel!)

### Hosting
For best results, use:
- LiteSpeed servers
- PHP 7.4 or higher
- HTTP/2 or HTTP/3
- SSD storage

## ❓ Troubleshooting

### Site Looks Wrong After Activation
- Go to Customizer → check all settings
- Click "Publish" to regenerate CSS
- Clear your browser cache (Ctrl+F5)

### Plugin Not Working
- Check UXNitro Performance settings
- Try disabling "Disable Block Library CSS"
- Try disabling "Disable REST API for Guests"
- Test with default WordPress theme to isolate

### Fonts Not Loading
- Check if "Disable Google Fonts" is enabled
- Try enabling it if fonts look wrong
- Use "Font Display Swap" for better performance
