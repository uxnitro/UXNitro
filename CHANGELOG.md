# Changelog

All notable changes to UXNitro will be documented in this file.

## [1.0.15] - 2026-04-08

### ✅ Added
- UXNitro Performance panel in WordPress Customizer
- WordPress cleanup tools (16 optimization options)
- Font optimization controls (4 settings)
- Media optimization (lazy loading for images & iframes)
- Database & heartbeat management
- Admin optimization settings
- Premium toggle switches with Active/Disabled labels
- All settings default to OFF for maximum safety

### 🔧 Changed
- Complete theme rebranding from Customify to UXNitro
- Theme version updated to 1.0.15
- All user-facing strings updated
- Admin dashboard rebranded
- Welcome page added
- Recommended plugins updated (The SEO Framework, W3 Total Cache)

### ❌ Removed
- All WooCommerce integration
- All pro/upsell features
- Pro Modules section from admin dashboard
- Font Awesome version selector from dashboard
- Cleanup Admin Bar setting
- Simplify Admin Menu setting
- Disable Update Nags setting
- Facebook community link (replaced with GitHub, Trustpilot, YouTube)

### 🎨 Improved
- Premium toggle switch styling with clear Active/Disabled states
- Divider lines between settings for better visual hierarchy
- Hover effects and smooth transitions
- Accessibility improvements (focus outlines)
- Community links section with modern design

---

## [1.0.0] - 2026-04-01

### ✅ Initial Release
- Forked from Customify 0.4.13
- Complete rebranding to UXNitro
- Theme name, author, and URLs updated
- Performance optimization panel created
- Modular architecture for custom features
- Admin welcome page added
- Documentation created

---

## Notes

- Internal class names (Customify, Customify_Customizer, etc.) preserved for backward compatibility
- All filter/action hooks preserved for child theme compatibility
- CSS class names unchanged to prevent breaking changes
- Text domain changed to `uxnitro` for proper translation support
